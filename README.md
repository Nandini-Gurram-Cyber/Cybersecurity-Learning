## 🔐 What is Cybersecurity?

Cybersecurity is the practice of protecting systems, networks, and data from:
- Unauthorized access  
- Cyber attacks  
- Data breaches  
- Damage or disruption  

## 🎯 Why Cybersecurity is Important?

- Protects personal and organizational data  
- Prevents financial losses  
- Ensures system availability and reliability  
- Maintains privacy and confidentiality  
- Reduces risk of cyber threats  

## 🧩 What Needs to Be Protected?

- 💻 **Systems** → Computers, servers, devices  
- 🌐 **Networks** → Internal & external communication  
- 📱 **Applications** → Software & web apps  
- 🗂️ **Data** → Personal, financial, organizational data  
- 🔑 **Credentials** → Passwords, usernames, access tokens  

## 🔎 Types of Cybersecurity

### 1. Network Security
Protects networks from unauthorized access and cyber attacks.  
**Example:** Firewalls, VPNs

### 2. Application Security
Secures software and applications from vulnerabilities.  

### 3. Endpoint Security
Protects devices like laptops, desktops, and mobile phones.  
**Example:** Antivirus, EDR tools  

### 4. Cloud Security
Protects data and applications stored in cloud environments like AWS, Azure.

### 5. Information Security
Protects sensitive data from unauthorized access or modification.

### 6. Identity and Access Management (IAM)
Controls user access to systems and resources based on roles and permissions.
